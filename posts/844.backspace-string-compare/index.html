844.backspace-string-compare

Description

力扣

给定 s 和 t 两个字符串，当它们分别被输入到空白的文本编辑器后，请你判断二者是否相等。# 代表退格字符。

如果相等，返回 true ；否则，返回 false 。

注意：如果对空文本输入退格字符，文本继续为空。

 

示例 1：

输入：s = "ab#c", t = "ad#c"
输出：true
解释：S 和 T 都会变成 “ac”。
示例 2：

输入：s = "ab##", t = "c#d#"
输出：true
解释：s 和 t 都会变成 “”。
示例 3：

输入：s = "a##c", t = "#a#c"
输出：true
解释：s 和 t 都会变成 “c”。
示例 4：

输入：s = "a#c", t = "b"
输出：false
解释：s 会变成 “c”，但 t 仍然是 “b”。
 

提示：

1 <= s.length, t.length <= 200
s 和 t 只含有小写字母以及字符 '#'
 

进阶：

你可以用 O(N) 的时间复杂度和 O(1) 的空间复杂度解决该问题吗？
 

Solution

  1. 在27.remove-element的基础上遇到 # 指针回退一步。然后截取有效区间。
  2. 在边界上思考了很久，更新ch[j-1]的值其实就是一种删除。
class Solution {
    public boolean backspaceCompare(String s, String t) {
        s = backspace(s);
        t = backspace(t);
        return s.equals(t);
    }
    public String backspace(String s) {
        int j = 0;
        char[] ch = s.toCharArray();
        for (int i = 0; i < ch.length; i++) {
            if (!"#".equals(String.valueOf(ch[i]))) {
                ch[j++] = ch[i];
            } else {
                if(j != 0) {
                    j--;
                }
            }
    	}
        return String.valueOf(ch).substring(0, j);
    }
}

Notice Points

  1. 字符串转数组：char[] ch = s.toCharArray();
  2. Char类型数组的值和String对比：String.valueOf(ch[i])
  3. String类型大小对比：s.equals(t), !"#".equals(s)
